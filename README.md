\# 🏰 Staffordshire Manor Solar Dashboard



A Streamlit-based dashboard to monitor live production data from my APSystems ECU Solar Array.



\## Features

\- \*\*Live Metrics:\*\* Displays current wattage and daily kWh yield.

\- \*\*Auto-Refresh:\*\* Updates every 2 minutes using Streamlit fragments.

\- \*\*Custom Styling:\*\* High-visibility metrics for dashboard tablets.



\## Setup

1\. Install requirements: `pip install -r requirements.txt`

2\. Add your `ECU\_IP` to `.streamlit/secrets.toml`

3\. Run the app: `streamlit run solar\_app.py`
